KeyValue Pipe

Using KeyValue Pipe to display object properties:

input json

{{ person | json }}

output

{{ item.key }}: {{ item.value }}

-------------

Slice Pipe

Using Slice Pipe to display a portion of an array:

input json

{{ numbers | json }}

output

{{ number }}

Custom Pipe | phoneNumberFormat

Using a custom pipe to format phone numbers:

1234567890

output

{{ '1234567890' | phoneNumberPipe:'US' }}

Custom Pipe | customPipeName

Using a custom pipe to format phone numbers:

{{numbers | json}}

output

{{ numbers | customPipeName:',' }}

Truncate Pipe | truncate

Using a custom pipe to format phone numbers:

{{"raj kumar singh" | json}}

output

{{ 'raj kumar singh' | truncate:5 }}

Filter Pipe | filter

{{persons | json}}

output

{{ person.name }} - {{ person.age }} - {{ person.city }}
